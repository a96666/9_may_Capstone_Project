LibTrack API Platform 📚

A Smart Library & Book Inventory Management System built using FastAPI, SQLite, SQLAlchemy, and Pandas.

This project was developed as part of a Capgemini Data Science Training Project.

🚀 Project Overview

LibTrack is a backend API platform that helps educational institutes manage:

Books
Students
Borrow/Return operations
Inventory tracking
CSV uploads
Reading analytics
Audit logs

The system replaces manual Excel-based management with a centralized API-based solution.

❌ Problems in Existing System

The institute was managing books manually using Excel sheets, which caused:

Duplicate book records
Missing inventory
No borrowing history
Difficult report generation
No centralized system
No APIs for future integration
